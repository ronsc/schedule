## Developer ##
``` 
$ git clone https://github.com/ronsc/schedule.git && cd schedule
```
```
$ npm install -g bower
```
```
$ npm install
```

* Run Webapp
```
$ npm start
```
> URL : [http://localhost:2222] (http://localhost:2222)

* Testing
```
$ npm test
```
